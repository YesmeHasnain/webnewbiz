## 10Web Copilot Features:

### 1. Chat-Based Editing
- Chat interface in bottom-right corner
- Natural language commands
- Click element + give instruction = AI edits that element
- "Selected element" chip shows in chat when element clicked

### 2. Text Editing
- Select text widget → "Write with AI" button
- Generate new text
- Improve existing content
- Adjust length (shorter/longer)
- Simplify language
- Fix grammar
- Change tone/style to match brand voice
- Rewrite entire sections

### 3. AI Image Generation
- Select image widget → "Generate Image"
- Describe desired image in text
- Choose style (photo, illustration, etc)
- Generate multiple variants to choose from
- Built-in (no external API needed)

### 4. Section Management
- Add new sections: "+Sections" button in top menu
- OR: Add container → "Add Section" button
- Select section type from list
- AI generates section content based on prompts
- Remove sections
- Reorder sections

### 5. Widget Management
- Add widgets (drag-drop)
- Duplicate widgets
- Remove widgets
- Edit widget settings via AI

### 6. Style Changes
- Change colors, fonts via chat
- Site-wide style changes
- Drag-and-drop style editing
- Refine copy and restructure sections

### 7. Element Selection
- Click on ANY element in preview
- "Selected element" prompt appears in chat
- Give instructions for that specific element
- AI modifies only that element

### 8. Undo
- Ask Co-Pilot to undo changes
- OR use Undo button in top ribbon
- Multiple undo steps

### 9. Code Editor
- View/edit code directly
- Code diff view (see what changed)

### 10. Publishing
- Publish button in top bar
- Preview before publishing
