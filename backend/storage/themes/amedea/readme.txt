The idea is to navigate a item-based listing with special effects on-hovered, it shows an image related to the item as preview.</br><strong>Animated Intro</strong>An interesting intro animation with image/video created with unique combination of motion, each ending in a circular look.</br><strong>Diagonal Media</strong>Animated slideshow for diagonal medias, tilted thumbnails and large titles those animate smoothly when navigating.</br><strong>Grid Hover Motion</strong>On-hover motion effects for image grids areas.</br><strong>Image Hover</strong>A collection of easy-to-use 4 main different category of total 34+ image hover elements ready for Elementor.</br><strong>Media Slider</strong>A collection of unique ideas for slideshow animations on fullscreen videos/images.</br><strong>On-Scroll Animated Grids</strong>Creative animated ideas for on-scroll effects using grids.</br><strong>On-Scroll Morph Effects</strong>A collection of easy-to-use the total 7+ image tilt/hover elements triggered by on-scroll ready for Elementor.</br><strong>Push Gallery</strong>Animated slideshow gallery for the grid items upon clicking the smooth pushing effect takes a place.</br><strong>Design Elements</strong>To improve your page design, ready-to-use page elements such as Contact/About widgets has been included with Accordion/Marquee/Tool-tips along with some other elements.</br>== PRO ==There are more than 100+ widgets available as PRO.Click <a href="https://amedea.pro/go/?p=pro" target="_blank">here</a> to see all of the widgets with demo websites and details.You can also check <a href="https://codecanyon.net/user/ki-themes/portfolio">Envato CodeCanyon</a> for the items are available for sale there.== Installation ==1.
